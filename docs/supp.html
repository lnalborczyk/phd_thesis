Chapter 9 Supplementary materials

Pre-registered protocol, open data, supplementary analyses as well as reproducible code and figures are available at https://osf.io/c9pag/.

Many packages have been used for the writing of this paper, among which the BEST, patchwork and ggplot2 packages for plotting (Kruschke & Meredith, 2018; Pedersen, 2017; Wickham et al., 2018), the tidybayes and sjstats packages for data analysis (Kay, 2018; Lüdecke, 2018), as well as the glue and tidyverse packages for code writing and formatting (Hester, 2017; Wickham, 2017).
